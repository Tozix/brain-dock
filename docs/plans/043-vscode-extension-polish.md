# 043 — VSCode-расширение: иконка мозга, i18n (ru/en), понятные ошибки

**Status:** Done
**Фаза:** Hosted product / client
**Дата:** 2026-06-10
**Связи:** [042-vscode-extension](042-vscode-extension.md)

## Проблема / запрос
1. Иконка в Activity Bar — обычный шестиугольник; нужна **иконка мозга**.
2. UI только на английском; нужна **поддержка ru/en + настройка-переключатель языка**.
3. При неверном `serverUrl` панель показывает `GET /projects → 404 Not Found` без подсказки —
   непонятно, что стучимся не на тот сервер (типичный кейс: API на `:3100`, дефолт `:3000`).

## Сделано / план
- [x] **Иконка мозга** — `media/icon.svg` (две полусферы, монохром `currentColor`).
- [x] **i18n** — `src/i18n.ts` (чистые словари en/ru + `t(lang, key, vars)`, 4 теста), настройка
  `brainDock.language` (`auto`/`en`/`ru`, `auto` = язык VS Code, `resolveLang()` в config). Панель,
  диалоги и сообщения локализованы; команды палитры — через `package.nls.json` / `package.nls.ru.json`.
  Проверено: панель рендерится на ru (ИНДЕКС/ЭКОНОМИЯ ТОКЕНОВ/Настроить агентов) и en.
- [x] **Понятная ошибка** — в состоянии ошибки панель показывает `serverUrl` + подсказку «проверьте,
  что URL указывает на brain-dock API» (типовой кейс 404: API на `:3100`, дефолт `:3000`).
- [x] **Версия** 0.1.0; `.vsix` пересобран (иконка + оба nls внутри).

## Definition of Done
- Иконка — мозг; переключение `brainDock.language` меняет язык панели и диалогов (ru/en); при 404
  видно, на какой `serverUrl` шёл запрос. `bun run ci` зелёный.
