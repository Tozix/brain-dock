## Цель
Полноценный self-service веб: регистрация → проект → индексация → API-ключ → подключение MCP —
без curl и без админа. Плюс админка (пользователи, аудит, usage) и продакшен-вход через
nginx на **хост-машине** (домен `brain-dock.ru`), всё остальное — в контейнерах.

## Решения
- **SPA**: Vite + React (latest stable) + react-router, без UI-китов — собственная дизайн-система
  (тёмная «инженерная консоль», кириллические шрифты, Bun-совместимая сборка). Русский UI.
- **Раздача**: контейнер `web` — статика Vite, отдаётся крошечным Bun-сервером (`:3300`,
  публикация только на `127.0.0.1`). TLS и маршрутизация — host-nginx.
- **Same-origin**: nginx проксирует `/` → web, `/api/v1` → api, `/mcp` → mcp; CORS не нужен.
- **Self-service API-ключи**: политика меняется — пользователь выпускает/отзывает СВОИ ключи
  (раньше — только ADMIN); админ видит и управляет всеми. Это требование полноценного веба.

## Этапы
- [x] **REST-расширения**: модуль `users` (админ: список, активация/деактивация, смена роли —
  SUPER_ADMIN, защита от самодеактивации); api-keys self-service (issue/list/revoke своих,
  админ — всех); `GET /usage` для админа по всем пользователям; OpenAPI + тесты.
- [x] **apps/web**: каркас (Vite, роутер, auth с refresh-flow, api-клиент), дизайн-система;
  страницы: вход/регистрация, дашборд проектов, проект (репозитории+статус индексации+загрузка
  папки, профиль, память/знания/документы, usage), API-ключи, «Подключить MCP» (готовый сниппет),
  админка (пользователи, аудит, usage).
- [x] **Контейнеризация**: `apps/web/Dockerfile`, сервис `web` в compose (профиль `app`),
  публикации api/mcp/web — только `127.0.0.1`.
- [x] **nginx (хост)**: `deploy/nginx/brain-dock.ru.conf` — 80→443, certbot TLS, `/`→web,
  `/api/v1`→api (client_max_body_size под upload-индексацию), `/mcp`→mcp (SSE: buffering off,
  длинные таймауты); инструкция в docs/deployment.
- [x] Документация (GUIDE §деплой/onboarding, ROADMAP) + `bun run ci` зелёный.

## Definition of Done
- Новый пользователь через браузер: регистрация → проект → загрузка папки → READY →
  свой ключ → скопировал сниппет → MCP работает.
- Админ видит пользователей/аудит/usage. nginx-конфиг применяется на хосте без правок.
